Hedges Chatter
With Hedges Chatter you can talk to others on different Discords, on some Topics!
Hedges Chatter is a Discord bot that syncs up channels between different servers and
· allows for secure communication between them.
· Cross server communication
· Public and Private Topics!
· In Private Topics you can Block servers
· Anti phishing included
· Get Reddit Memes and share it to the Meme Topic :)
· For Setup Help use /info or go to the Support Server and ask there!
Permissions For The Bot
→ Create Invite
→ Embed Links
→ External Emojis / Sticker
→ Attach Files
→ Add Reaction
→ Read / Send Messages
→ Send Messages in Threads
→ Manage/Create Public/Private Threads
→ Read message history
